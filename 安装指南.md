# OptiVerse - 安装和运行指南

## 问题说明

您当前的 Miniforge3 安装已损坏，缺少核心的 Python 文件，需要重新安装 Python 环境。

## 快速解决方案（推荐）

### 方法 1：使用 Python 官网（最简单）

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本（选择 Windows installer (64-bit)）
3. 运行安装程序时，请确保**勾选 "Add Python to PATH"**
4. 安装完成后，重新打开终端，运行以下命令：

```bash
cd "d:\桌面\OptiVerse — 交互式数值优化演武场"
pip install streamlit matplotlib plotly sympy numpy
streamlit run app.py
```

### 方法 2：重新安装 Miniforge3

1. 访问 https://github.com/conda-forge/miniforge/releases
2. 下载 Miniforge3-Windows-x86_64.exe
3. 安装到默认位置（C:\Users\lenovo\miniforge3）
4. 在开始菜单打开 **"Miniforge Prompt"**
5. 在该终端中运行以下命令：

```bash
conda create -n optiverse python=3.10
conda activate optiverse
pip install streamlit matplotlib plotly sympy numpy
cd "d:\桌面\OptiVerse — 交互式数值优化演武场"
streamlit run app.py
```

### 方法 3：使用 PyCharm（如果已安装）

1. 打开 PyCharm
2. 打开项目文件夹 `d:\桌面\OptiVerse — 交互式数值优化演武场`
3. PyCharm 会提示创建新的 Python 解释器（选择 "New Virtual Environment"）
4. 在 PyCharm 的 Terminal 中运行：
   ```bash
   pip install streamlit matplotlib plotly sympy numpy
   ```
5. 右键点击 `app.py` 选择 "Run 'app'"

## 验证安装成功

安装完成后，您应该能在浏览器中看到：
- 应用标题 "OptiVerse - 交互式数值优化演武场"
- 左侧侧边栏可以选择测试函数
- 中间显示等高线图
- 可以使用"随机起点"按钮

如果遇到问题，请检查终端中的错误信息。
